# Rock-paper-Scissors

A simple Rock Paper Scissors game built with JavaScript, played entirely through the browser console. This project is part of a beginner JavaScript learning exercise and focuses on core concepts like functions, conditionals, loops, and user input.

# Feature
- Play against computer
- choose how many round you want to play
- Randomized computer choices
- User input collected via computer browser `prompt`
- Score tracking for both player

## Technologies Used

- HTML
- JavaScript
- Browser Console (DevTools)
